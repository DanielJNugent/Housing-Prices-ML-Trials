# Project Overview
This is an Original Project with 3 trials measuring the correlation between different variables and house prices in diffrent areas
This is a beginner machine learning project. This includes setting up a model and measuring error. This shows how to build an end-to-end machine learning project using Python and Jupyter.

This is done by using data from AUS, NYC, and IOWA. The goal was to predict the price/saleprice of a house based on historical and current data.

**Project Steps**

1. Find and explore the data.
2. (If necessary) Reshape the data to predict your target.
3. Clean the data for ML.
4. Pick an error metric.
5. Split your data.
6. Train a model.

File overview:

* `AUS_REALESTATE.csv` - The Excel File containing the Housing Data for AUS
* `Housing_prices_TRIAL_1.ipynb` - The Jupyter file for AUS (TRIAL 1)
* * `Trial_2_Housing_Prices.csv` - The Excel File containing the Housing Data for NYC
* `Housing_prices_TRIAL_2.ipynb` - The Jupyter file for NYC (TRIAL 2)
* * `Trial_3_Housing_Prices.csv` - The Excel File containing the Housing Data for IOWA
* `Housing_Prices_TRIAL_3.ipynb` - The Jupyter file for IOWA (TRIAL 3)
